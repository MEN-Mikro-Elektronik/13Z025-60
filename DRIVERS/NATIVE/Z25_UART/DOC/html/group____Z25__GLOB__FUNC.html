) 
 

Routine to install driver in the VxWorks I/O system.

This routine is used to install the driver after the boot sequence.
Z25_CreateDevice("/tyZ25", "0x1e 0x0e",1,0,0x80,0,intConnect,inEnable)
-> "tyZ25", e.g. for device name string
-> "0x1e 0x... 0x0e", 0x1e and following=device number of bridges and 0x0e=device number of desired PCI device
If intConnectP or intEnableP is zero, the default VxWorks functions pciIntConnect and intEnable are used.

Parameters:
devNameP serial device name
pathStringP PCI path of device
usePciIrq use PCI irq
irqBase interrupt base
irqOffset BSP specific irq offset for chameleon V2 FPGAs for chameleon V1 devices always zero
uartFreq uart frequency base, if 0 DOS compatibility freqency (1.8432MHz)
intConnectAddr interrupt connect function pointer
intEnableAddr interrupt enable function pointer
Returns:
Z25_HDL - Z25 handle

Returns:

u_int16  unit,
u_int16  channel
) 
 

Routine to initialize the driver at boot time.

This routine is used to make the BSP initialization of the driver. It must be called in sysSerialHwInit2. Before using this function call Z25_InitDriver and Z25_FindUartUnits to obtain a valid Z25 handle and the available UART units.

Parameters:
hdlP Z25 handle returned by Z25_Init
unit number of 16Zx25 unit
channel number of channel in unit
Returns:
Z25_OK or Z25_ERROR
